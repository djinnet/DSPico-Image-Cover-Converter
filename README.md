# DSPico-Image-Conversor

Convierte casi cualquier imagen en el fortado de cover de DSPico
Convert almost any image to DSPico Conver Format


Para compilar usa .Net9 o modifica el .csproj para usar la versión que te plazca

https://dotnet.microsoft.com/es-es/download/dotnet/9.0



Requiere Magisck para funcionar correctamente

https://imagemagick.org/#gsc.tab=0 - Un saludo a su gra nesfuerzo -
