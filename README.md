# DSPico-Image-Conversor

Convierte casi cualquier imagen en el fortado de cover de DSPico

Convert almost any image to DSPico Conver Format


Escrito en C++

Write in C++


Para compilar usa w64devkit

To compile use w64devkit

https://github.com/skeeto/w64devkit


Versión Española

g++ PicoLauncherESP.cpp resources.o -o PicoLauncherESP.exe -mwindows -lcomctl32 -lwinmm -lole32 -luuid -lgdiplus -lshlwapi -static -Os -s


English Version

g++ PicoLauncherENG.cpp resources.o -o PicoLauncherENG.exe -mwindows -lcomctl32 -lwinmm -lole32 -luuid -lgdiplus -lshlwapi -static -Os -s


Requiere Magick instalado y en el Path para funcionar correctamente

Magick is required to be installed and in the PATH to function correctly


Link Magick
